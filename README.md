# Voice to Text

## Getting Started

Get the API key from [Deepgram](https://deepgram.com/)

Add the API key to the .env file

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
